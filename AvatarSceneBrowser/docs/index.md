# Avatar Scene Browser 插件文档

!!! info "插件简介"
    Avatar Scene Browser 的初始目的是一款面向 Unity Editor 扩展。它通过缩略图、标签系统，帮助您在拥有数百个场景的大型项目中也能一眼找到、打开并批量管理需要的场景与 Avatar 预制体。

## 主要功能

- **化身模型预览与管理** - 支持多种格式的化身导入和组织
- **场景快速切换与编辑** - 高效的场景管理和预览系统
- **自定义化身属性配置** - 灵活的化身参数调整
- **批量处理工具集** - 提高工作效率的批处理功能
- **实时预览与调试** - 所见即所得的编辑体验
